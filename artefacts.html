Core Concepts

Experiment Artefacts

Every run is fully self-contained. Here's exactly what gets created and where.

Directory Structure

Every run creates a single timestamped directory under log_dir/. Nothing is scattered — all outputs live together:

logs/
  └── exp_20250502_143022/  # <experiment_name>_<timestamp>
      ├── pipeline.log                 # structured log of every pipeline event
      ├── preprocessing_report.md      # profile, missing data, skewness, steps used
      ├── preprocessed_train.csv       # training data after preprocessing
      ├── preprocessed_test.csv        # test data after preprocessing
      ├── model_metrics.xlsx           # 3 sheets: All Models · Ranked · Failed
      ├── selection_report.md          # selector config, leaderboard, candidates
      ├── tuning_report.md             # AI usage, search spaces, trial history
      └── models/
          ├── preprocessing_pipeline.joblib
          └── RandomForestClassifier.joblib

File Descriptions

pipeline.log

Structured log of every event in the pipeline at INFO / WARNING / ERROR levels. Includes LLM retry attempts and fallback notices.

preprocessing_report.md

Dataset profile summary, missing data analysis with severity indicators, skewness measurements, and the list of preprocessing steps applied.

preprocessed_train.csv

The training split after the full preprocessing pipeline has been applied. Useful for debugging or offline analysis.

preprocessed_test.csv

The test split after preprocessing, transformed using the fitted pipeline (not re-fitted on test data).

model_metrics.xlsx

Three-sheet Excel workbook: All Models (every trained model), Ranked (sorted by primary metric, best row highlighted), and Failed (models that errored).

selection_report.md

Which selection strategy was used, the full leaderboard, the selected candidates forwarded to the optimizer, and a summary.

tuning_report.md

AI Advisor usage summary (token counts, fallback status), generated search spaces per model, full trial history with medals, and the best configuration found.

models/

The fitted preprocessing pipeline and the retrained best model, both serialised with joblib for fast loading at inference time.

Running Inference with Saved Artefacts

Load the saved pipeline and model to make predictions on new data:

import joblib, pandas as pd

pipeline = joblib.load("logs/exp_20250502_143022/models/preprocessing_pipeline.joblib")
model    = joblib.load("logs/exp_20250502_143022/models/RandomForestClassifier.joblib")

X_new = pd.read_csv("new_data.csv").drop(columns=["target"])
preds = model.predict(pipeline.transform(X_new))
✅
Always apply the pipeline first pipeline.transform(X_new) must be called before model.predict(). The pipeline handles all encoding and scaling that the model expects.
