Usage

CLI Reference

Run MLTuneX from the terminal — no Python script required.

Examples

Minimal run with AI tuning

mltunex --data titanic.csv \
        --target Survived \
        --task classification \
        --llm "Groq:qwen/qwen3-32b"

Skip tuning — no API key needed

mltunex --data fraud.csv \
        --target label \
        --task classification \
        --no-tune \
        --selector stability \
        --top-k 5

Regression with parallel training

mltunex --data housing.parquet \
        --target SalePrice \
        --task regression \
        --llm "OpenAI:gpt-4o" \
        --parallel \
        --jobs 8 \
        --trials 50 \
        --selector topk \
        --top-k 3 \
        --exp-name housing_run1

Launch the Streamlit UI

mltunex ui

Full Options Reference

Required Arguments
--data PATH
required
Dataset file path. Accepts .csv, .xlsx, .parquet, .feather.
--target COLUMN
required
Name of the target column to predict.
--task TYPE
required
classification or regression.
--llm PROVIDER:MODEL
required unless --no-tune
LLM for AI tuning. Format: Provider:ModelName. Examples: "Groq:qwen/qwen3-32b", "OpenAI:gpt-4o". Not required when --no-tune is set.
Training Options
--test-size FLOAT
Test split fraction. Default: 0.2.
--no-preprocess
Skip adaptive preprocessing. Data is passed to models as-is.
--parallel
Train models in parallel using a multiprocessing pool.
--jobs N
Worker count for parallel training. -1 uses all available CPUs. Default: -1.
--library LIB
Model library backend. Default: sklearn.
Tuning Options
--no-tune
Skip AI hyperparameter optimisation entirely. Top-K models are selected and saved directly. --llm is not required.
--trials N
Optuna trial count. Default: 25.
--optimizer NAME
Optimizer backend. Default: optuna. Extendable via OptimizerFactory.register().
Model Selection
--selector NAME
topk | stability | generalization. Default: topk.
--top-k N
Number of candidates forwarded to the optimizer. Default: 3.
--primary-metric NAME
Override the primary ranking metric. Defaults to Accuracy for classification, R2 for regression.
--stability-weight F
Instability penalty weight for the stability selector. Default: 0.2.
--train-metric NAME
Train-set metric column name for the generalization selector.
--gap-penalty F
Train/test gap penalty weight for the generalization selector. Default: 0.5.
--profiling TYPE
basic | extended. Default: extended.
Output
--results DIR
Results CSV directory. Default: results/.
--log-dir DIR
Experiment log root. Default: logs/. All artefacts land inside a timestamped subdirectory here.
--exp-name NAME
Tag prepended to the experiment folder name. Default: exp. e.g. housing_v1 → housing_v1_20250502_143022/.
⚠️
--llm is required when AI tuning is enabled Either provide --llm "Groq:qwen/qwen3-32b" or add --no-tune to skip AI tuning entirely.
