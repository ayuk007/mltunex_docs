Getting Started

Quick Start

Run your first automated ML pipeline in under 5 minutes.

Your First Run

This is everything you need. Three lines of Python, and MLTuneX handles the rest:

from mltunex.orchestrator import MLTuneXOrchestrator, OrchestratorConfig

config = OrchestratorConfig(
    source                    = "titanic.csv",
    target_column             = "Survived",
    task_type                 = "classification",
    model_provider_model_name = "Groq:qwen/qwen3-32b",
)

MLTuneXOrchestrator(config).run()

What happens when you call .run()

1

Data Loading & Profiling

Reads your CSV and profiles every column — types, missing values, cardinality, skewness.

2

Adaptive Preprocessing

Builds and fits a preprocessing pipeline based on the profile: imputation → outlier clipping → encoding → scaling.

3

Train All Models

Trains every applicable sklearn / XGBoost / LightGBM model and displays a live progress table in the terminal.

4

Evaluate & Select Top-K

Evaluates each model on the held-out test set and selects the top 3 candidates.

5

AI-guided Search Spaces

Sends the dataset profile to Groq, which returns Optuna-compatible hyperparameter search spaces tailored to your data.

6

Optuna Optimisation

Runs 25 Optuna trials per candidate, retrains with the best params, saves model + pipeline as .joblib files.

7

Artefacts Written

A timestamped experiment folder is created with Markdown reports, Excel metrics, and a structured pipeline log.

No API Key? Use No-tune Mode

Set tune_models=False to skip the LLM and Optuna step entirely. Top-K models are selected and saved directly:

config = OrchestratorConfig(
    source        = "data/fraud.csv",
    target_column = "is_fraud",
    task_type     = "classification",
    tune_models   = False,
    top_k         = 3,
)
MLTuneXOrchestrator(config).run()

Using the CLI

mltunex --data titanic.csv \
        --target Survived \
        --task classification \
        --llm "Groq:qwen/qwen3-32b"

Using the Streamlit UI

mltunex ui

Opens at http://localhost:8501 — upload your dataset, configure everything visually, and launch the run from a browser.

📂
Where are my results? All outputs land in logs/exp_<timestamp>/ by default. See the Experiment Artefacts page for the full directory structure.
