Getting Started

Installation

Get MLTuneX installed in your environment in under a minute.

Requirements

🐍

Python 3.8+

Compatible with Python 3.8 through 3.12.

📦

pip

Standard package installer included with Python.

🔑

API Key (optional)

Only needed when using AI-guided hyperparameter tuning.

From PyPI Recommended

pip install MLTuneX

From Source

Useful for contributing or testing the latest development version.

git clone https://github.com/ayuk007/MLTuneX.git
cd MLTuneX
pip install -e .

Optional Extras

Install optional dependencies to unlock additional features:

ExtraCommandWhat it unlocks
catboostpip install MLTuneX[catboost]CatBoost model support
parquetpip install MLTuneX[parquet]Parquet / Feather file reading

Setting Up API Keys

API keys are only required when tune_models=True (the default). You can skip this entirely with tune_models=False.

Groq (free tier available)

# Linux / macOS
export GROQ_API_KEY="gsk_..."

# Windows
set GROQ_API_KEY=gsk_...

OpenAI

# Linux / macOS
export OPENAI_API_KEY="sk-..."

# Windows
set OPENAI_API_KEY=sk-...
🔐
Streamlit UI alternative You can also paste API keys directly in the Streamlit UI — they are used only in the current session and never written to disk.
✅
No-key mode Set tune_models=False in OrchestratorConfig to run the entire pipeline without any LLM or API key. Top-K models will be selected and saved directly.

Verify Installation

python -c "import mltunex; print('MLTuneX installed successfully')"
mltunex --help
