Automated ML
Fine-Tuning

MLTuneX takes your dataset from raw CSV to a fine-tuned model in a single command — no boilerplate, no guesswork.

⚡ Get Started 📖 API Reference
Raw dataset (CSV / Excel / Parquet / DataFrame)
  │
  ▼
[1] Profile → detect types, missing values, skewness, cardinality
  │
  ▼
[2] Preprocess → adaptive imputation · encoding · scaling · outlier clipping
  │
  ▼
[3] Train all models → sklearn · XGBoost · LightGBM  (parallel optional)
  │
  ▼
[4] Evaluate + Select top-K → Accuracy · F1 · R² · RMSE …
  │
  ▼
[5] AI Advisor → Groq / OpenAI generates smart Optuna search spaces
  │
  ▼
[6] Optuna optimises → best model + fitted pipeline saved
  │
  ▼
Experiment artefacts → Markdown reports · Excel metrics · pipeline.log · token usage

Features

🚀

Zero-boilerplate AutoML

One function call covers the full pipeline end to end.

🧠

AI-guided Tuning

LLM-generated hyperparameter search spaces tailored to your dataset profile.

🔌

Pluggable LLM Registry

Groq and OpenAI built-in; attach any custom LLM in ~10 lines.

🛡️

Schema-validated AI Output

3-retry loop with exponential back-off and predefined fallback for 28 models.

📊

Token Usage Tracking

Prompt / completion / total tokens logged per run in the tuning report.

⚗️

Adaptive Preprocessing

Profile-driven pipeline built automatically — imputation, encoding, scaling.

⚡

Parallel Training

Optional multiprocessing pool to train all models simultaneously.

📓

Jupyter / Colab Support

Auto-detects runtime and uses plain output in notebooks.

🎛️

3 Interfaces

Python API · CLI (mltunex) · Streamlit UI (mltunex ui)

📁

Experiment Artefacts

Every run writes to a single timestamped directory with reports, models, and logs.

🏗️

Pluggable Selectors

TopK, StabilityAware, GeneralizationAware, or register your own.

🚫

No-tune Mode

Skip the LLM entirely — select and save top-K models without an API key.

Install in 30 seconds

pip install MLTuneX
💡
No API key needed to start Set tune_models=False to run the full pipeline without any LLM — just top-K model selection and saving.
